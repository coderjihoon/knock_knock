# AGENTS.md

## Project intent
This project is a premium Korean community web app for "Knock Knock".
The brand should feel warm, thoughtful, calm, editorial, and quietly premium.

## Design rules
- Always design mobile-first unless explicitly told otherwise.
- Prioritize vertical rhythm and whitespace.
- Typography should feel premium and emotionally resonant.
- Avoid corporate SaaS UI patterns.
- Avoid overly dense cards.
- Prefer title -> image -> body flow for storytelling sections.
- Use warm olive + beige tones only.

## Brand palette
- bg-main: #FBF8F3
- bg-soft: #F2EDE3
- card-white: #FFFFFF
- light-olive: #E8EDE1
- main-olive: #A3AE7C
- deep-olive: #6F7D4E
- dark-olive: #48553A
- text-primary: #2B2A26
- text-secondary: #6B665E
- accent-brown: #5A4A3C

## Typography
- Title font: Hahmlet
- Body font: Pretendard

## Commands
- Run dev server: npm run dev
- Run lint: npm run lint

## Engineering rules
- Use Next.js App Router
- Use Tailwind CSS
- Keep components reusable
- Prefer clean semantic HTML
- Do not add unnecessary dependencies

## Done means
- The homepage renders without errors
- The feature section is mobile-first
- The layout is visually clean on a 390px-wide screen
- Lint passes
